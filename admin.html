Panduan Penggunaan Dashboard

📖 Cara Memulai

Dashboard ini terhubung langsung ke database Supabase. Setiap perubahan yang Anda simpan akan langsung tampil di website Braveline secara real-time.

🔐

1. Login & Keamanan

  • Masukkan password untuk akses dashboard.
  • Session aktif selama 24 jam.
  • Ganti password di menu Pengaturan → Keamanan.
  • Klik Logout di sidebar untuk keluar.
✏️

2. Edit Teks & Konten

  • Pilih seksi di sidebar kiri (misalnya Seksi Hero).
  • Ubah isian teks sesuai kebutuhan.
  • Klik tombol SIMPAN untuk menyimpan perubahan.
  • Perubahan langsung tampil di website.
🖼️

3. Upload Foto

  • Klik kotak foto untuk memilih gambar dari komputer.
  • Foto otomatis dikompres (max 800×800px, format WebP).
  • Mendukung JPG, PNG, dan WebP.
  • Berlaku di Katalog, Carousel, dan Testimonial.
📋

4. Tambah / Hapus Data

  • Klik tombol + Tambah untuk menambah item baru.
  • Klik tombol EDIT pada item untuk mengubahnya.
  • Klik tombol HAPUS untuk menghapus item (tidak bisa dikembalikan).
⚙️

5. Pengaturan Global

  • Site Title — nama website di tab browser.
  • WhatsApp — nomor WA untuk tombol order.
  • Footer — tagline, deskripsi, alamat, IG, TikTok.
  • Setelah ubah, klik SIMPAN SETTINGS.
💡

6. Tips Penting

  • Gunakan Urutan (Angka) untuk mengatur urutan tampil item.
  • Angka urutan lebih kecil = tampil lebih awal.
  • Centang Unggulan pada paket harga untuk highlight khusus.
  • Gunakan Lihat Website → di header untuk preview hasil.
